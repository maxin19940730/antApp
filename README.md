一个简易的新闻站点(dva版)

---

### 技术栈:

- react
- dva
  - react-router
  - redux
  - redux-saga
  - fetch
- antd design  

### 脚手架工具:

- dva-cli  

## 运行方式:

1、yarn 启动  
  - yarn install
  - yarn run start  
  
2、npm 启动  
  - npm install
  - npm run start
  
> 提示：切换分支查看不同版本
